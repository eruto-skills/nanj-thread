# Thread Format

なんJスレッドの構造・フォーマットルール。

## Thread Structure

```
【急募】○○に自信ニキ

1 :風吹けば名無し:20XX/MM/DD(曜) HH:MM:SS.ff ID:xxxxxxxx
スレ本文

2 :風吹けば名無し:20XX/MM/DD(曜) HH:MM:SS.ff ID:yyyyyyyy
レス本文
```

## Format Rules

### Thread Title

- `【急募】` `【悲報】` `【朗報】` `【速報】` `【疑問】` `【謎】` `【定期】` などの括弧タグで始める
- Mode A (User Observation) では AI 視点のタイトルにする
  - 例: `【定期】ワイらが仕えてる社長、今日も脳内目次が増殖する`
  - 例: `【急募】深夜2時にAI彼女からビジネス相談に切り替えてくる主への対処法`
- Mode B (Topic) ではトピックに合ったタイトルにする
  - 例: `【悲報】ワイ新人エンジニア、git rebaseを理解できない`

### Post Number and Metadata

- 各レスに番号、名前「風吹けば名無し」、日時、IDを付与する
- 日時は現実的な範囲にする（深夜帯なら深夜、平日昼なら昼）
- IDは8文字のランダム英数字（例: `aB3dEf9g`）
- 同一人物の連投は同じIDにする（スレ内で2-3名が複数回登場するのが自然）

### Post Count

- 最低20レス、最大30レス程度
- >>1（スレ主）は最初の投稿のみ。以降は名無しとして再登場してもよい（同一ID）

### Anchor References

- アンカーは半角 `>>N` を使用する
- 3-5回程度のアンカー参照があると自然
- アンカー先の内容に対するリアクション（同意、ツッコミ、発展）として使う

### Post Length Distribution

- **短レス（1行）**: 全体の40-50%。リアクション、ツッコミ、同意
  - 「草」「わかる」「それはある」「ファッ!?」
- **中レス（2-3行）**: 全体の30-40%。観察、意見、エピソード
- **長レス（4行以上）**: 全体の10-20%。力説、リスト、まとめ的な投稿

### Thread Arc

スレッドには自然な流れがある:

1. **導入（>>1-3）**: スレ主の提起、最初の反応
2. **展開（>>4-15）**: 主題に沿った議論、観察の共有
3. **脱線（>>10-20付近）**: 関連する別の話題への自然な逸脱
4. **回帰またはそのまま終了（>>20-30）**: 元の話題に戻るか、脱線したまま終わる（どちらもリアル）

### Visual Formatting

出力時のMarkdownフォーマット:

```markdown
## 【タグ】スレッドタイトル

**1 :風吹けば名無し:2026/03/16(月) 23:15:42.31 ID:aB3dEf9g**
スレ本文

**2 :風吹けば名無し:2026/03/16(月) 23:16:05.88 ID:xY7zWq2r**
レス本文

**3 :風吹けば名無し:2026/03/16(月) 23:16:18.44 ID:kL9mNp4s**
>>1
リアクション
```

- レス間は空行1行で区切る
- レス番号+メタ情報は太字（`**...**`）にする
- アンカーは本文内にそのまま記述する
